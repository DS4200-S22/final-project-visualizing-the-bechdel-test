Visualizing the Bechdel Test

Blake Heyman, Sisira Mandapaka, Zoe Morris, Faith Occhipinti

Course Project as part of DS4200 S22: Information Visualization, taught by Prof. Ab Mosca, Data Visualization @ Khoury, Northeastern University.

Abstract

The Bechdel Test has been thought of as an informal rule in film for some time now: a film passes this test if two named women have a conversation with each other about something other than a man. We aim to create a visualization tool for the purpose of examining biases in film, using the Bechdel Test as a way to quantitatively measure them. Our visualization tool aims to provide information on the correlation between a movie's Bechdel Test status and factors such as box office success, year of release, and general audience sentiment using D3.

Visualization Explanation

Our visualization primarily functions as a comparison and general overview tool, providing multiple different components so users can understand the correlation between different facets of film and representation of women in these films. \ The line chart at the top will be a sticky header that allows for the user to brush over a range of years, which will then automatically update the year range of every visualization on the page, focusing on that particular subset of data. This allows them to compare the breakdown of movies that pass and fail the Bechdel test over the years, both in the past and in more contemporary movies. When no years are selected, every visualization defaults to representing all movies in the dataset. Within the segmented bar chart component of our visualization, the user can select a number of genres featured in our dataset from a dropdown menu on the right of the plot. This allows the user to be able to compare the movies that pass and fail certain tests by genre so they can see the disparity in gender equality and representation by genre. Some genres may have better representation than others, and this may have changed over time. Each segment of the bar chart represents the breakdown of how many criteria of the Bechdel Test the given movie passes. The user can hover over a segment to view a tooltip that explains the percentage of movies in that segment, as well as the specific test that is passed in that segment. Meanwhile, within the violin plot, the user can select the metric they want to see the distribution of over the dataset (either the IMDb rating of the movie or the runtime of the movie). This allows the user to view audience sentiment toward movies that pass and fail the Bechdel test in that time period when IMDb rating is selected as the metric. They can also view the relationship between runtime and pass/fail rates over time. Users can hover over the box and whisker plot in the middle of each violin to view a tooltip that gives summary statistics for each violin. Include revised explanation and presentation slides!
